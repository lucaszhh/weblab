¿Qué hacemos en WEB LAB?

En este taller exploramos herramientas de código abierto y de uso gratuito para crear, diseñar y programar páginas web con los lenguajes HTML, CSS y JavaScript.

Nuestro enfoque combina arte, diseño y programación para desarrollar experiencias digitales.
